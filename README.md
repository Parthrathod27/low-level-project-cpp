# Low-Level C++ Projects for Beginners

This repository contains **beginner-friendly low-level C++ projects** that demonstrate the use of **basic C++ libraries** and **OOP concepts**. Each project is a small console-based application designed to help beginners learn C++ programming in a practical way.

---

## Projects Included

1. **SBI Digital Banking System**  
   - A simple console-based banking system with account creation, deposit, withdrawal, and balance checking.
   
2. **Student management**  
   - A simple Student Management System built in C++ that allows users to add, view, search, update, and delete student records using file handling..

3. **[Tic Tac Toe]**  
   - A simple tic tac toe game built in c++ played between 2 players .

- Console-based programs (no GUI required).
